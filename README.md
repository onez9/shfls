### Небольшой файловый сервер с фозможностью просмотра и скачивания видео.

Что это за проект:
- чат
- просмотр и загрузка видео
- файловое хранилище

#### Установка:
```bash
git clone https://github.com/onez9/shfls.git
cd shfls
npm i
```

#### Запуск:
```python
python3 start.py
```
#### Или (обе команды в разных терминалах):
```bash
npm run dev
npm start
```
